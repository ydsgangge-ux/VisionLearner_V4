# 快速开始 - 打开 test_ui.html

## 步骤 1: 启动服务器

在项目目录中打开命令行，运行以下任一命令：

### 方法 1: 使用批处理文件（推荐）
双击运行 `start_server.bat`

### 方法 2: 使用 Python 命令
```bash
python restart_server.py
```

### 方法 3: 使用原始启动命令
```bash
python main.py --web --port 5000
```

您会看到类似这样的输出：
```
+============================================================+
|  [WEB] VisionLearner Web UI Started                        |
+------------------------------------------------------------+
|  Local:   http://localhost:5000                           |
|  Network: http://192.168.x.x:5000                         |
|  Make sure phone is on same WiFi                          |
+============================================================+
```

## 步骤 2: 打开测试界面

### 方法 1: 直接打开文件（推荐用于测试）

直接在浏览器中打开 `test_ui.html` 文件：

1. 在文件资源管理器中找到 `test_ui.html`
2. 右键点击 → "打开方式" → 选择浏览器（Chrome、Edge、Firefox 等）
3. 或者直接双击文件

**注意**: 直接打开文件时，浏览器地址栏会显示 `file:///d:/AGI/VisionLearner%20-1/test_ui.html`

### 方法 2: 通过主界面访问

如果服务器的主界面 (`visionlearner_ui.html`) 已经打开，可以：

1. 在浏览器地址栏输入：`http://localhost:5000`
2. 这会打开主界面
3. 如果想打开测试界面，可以手动在地址栏改为：
   ```
   file:///d:/AGI/VisionLearner%20-1/test_ui.html
   ```

## 测试界面功能

`test_ui.html` 提供了以下功能卡片：

1. **系统状态** - 测试 `/api/status`
2. **目标列表** - 获取所有目标
3. **选择/删除目标** - 切换或删除目标
4. **学习进度** - 查看当前目标进度
5. **思维导图** - 获取思维导图数据
6. **AI问答** - 测试问答功能
7. **创建目标** - 创建新的学习目标
8. **技能列表** - 查看可用技能

## 使用示例

1. 点击"测试 /api/goals"按钮获取所有目标
2. 在"选择目标"下拉框中选择一个目标
3. 点击"切换到该目标"来激活该目标
4. 或者点击红色的"删除目标"按钮来删除它

## 注意事项

- 确保服务器正在运行（端口 5000）
- 如果遇到 CORS 错误，确保从服务器访问而不是直接打开文件
- 直接打开文件时，API 请求会发送到 `http://localhost:5000/api/*`

## 停止服务器

在运行服务器的命令行窗口中按 `Ctrl + C` 停止服务器。
